SMS Sort

SMS Sort is a demo that combines AngularJS v1.2.23, Bootstrap v3.3.4, and Font Awesome v4.7.0 to make a user accessible and locally contained way to explore SMS messages.

Thanks to cipsum.com, nietzsche-ipsum.com, officeipsum.com, and metaphorpsum.com for message text.
